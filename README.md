**Amarire Theme** is a modern, lightweight, secure, and developer-friendly WordPress starter theme optimized for **Elementor**, **WooCommerce**, **Gutenberg**, and major plugins.

- **Performance & Speed:** Ultra-lightweight, optimized asset loading, minimal markup, SEO-friendly.  
- **Security:** Follows WP security best practices, sanitization, escaping, safe structure.

- **WooCommerce Ready:** Shop pages, product templates, cart, checkout fully supported.
